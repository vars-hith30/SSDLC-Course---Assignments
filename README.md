# SSDLCP Course Assignments

## Overview

This repository contains the assignments and practical exercises completed during the **Secure Software Development Life Cycle Professional (SSDLCP)** course conducted by **CDAC (Centre for Development of Advanced Computing)**.

The assignments cover secure software development practices, secure coding principles, vulnerability identification, threat analysis, and software security concepts learned throughout the course.

---

## Topics Covered

- Secure Software Development Life Cycle (SSDLC)
- Secure Coding Practices
- Threat Modeling
- Risk Assessment
- Vulnerability Analysis
- Authentication and Authorization
- Cryptography Basics
- Web Application Security
- Secure Design Principles
- Security Testing


---

## Purpose

This repository serves as a record of the coursework completed during the SSDLCP training and demonstrates the practical application of secure software development concepts.
